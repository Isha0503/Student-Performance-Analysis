# 🎓 Student Performance Dashboard

An interactive Student Performance Dashboard built using **Python, Streamlit, Pandas, Matplotlib, Seaborn, and Plotly**.

This dashboard helps analyze student academic performance through interactive visualizations and filtering options.

---

## 📌 Features

- 📊 Dashboard Summary
  - Total Students
  - Average Grade
  - Highest Grade
  - Pass Percentage
  - Fail Percentage

- 🎛️ Interactive Sidebar Filters
  - Filter by Gender
  - Filter by School

- 📈 Interactive Charts
  - Gender Performance (Plotly)
  - Pass vs Fail Donut Chart (Plotly)
  - Grade Distribution Histogram
  - Internet Access Analysis
  - Correlation Heatmap

- 🏆 Top 10 Students Table

- 📥 Download Filtered Dataset (CSV)

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Isha0503/Student_Performance_Analysis.git
```

Go to project folder

```bash
cd Student_Performance_Analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
